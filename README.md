
# TryHackMe Writeups

This repository contains my personal **writeups for rooms and machines from the TryHackMe platform**.

The main goal of this repository is to document my learning process while practicing different areas of cybersecurity and penetration testing.

---

# Topics Covered

While solving TryHackMe rooms I practice:

- Network enumeration
- Web exploitation
- Linux privilege escalation
- Windows privilege escalation
- Active Directory attacks
- Basic and intermediate CTF techniques

---

# Repository Structure

Each room has its own directory containing the writeup and notes.

Example:

```
tryhackme-writeups
│
├── README.md
│
├── room-name
│   └── writeup.md
│
├── another-room
│   └── writeup.md
```
